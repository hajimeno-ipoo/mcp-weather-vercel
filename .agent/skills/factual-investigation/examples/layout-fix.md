# 事実調査スキルの活用例: レイアウト崩れの解決

## 状況
ウィジェットで要素を `display: flex` に設定して `innerHTML` で挿入したが、実際には縦に並んでしまう。

## スキルを適用した行動
1. **証跡の収集**: `browser_subagent` を起動し、ウィジェット内の要素を `getComputedStyle` で検査。
2. **事実の観測**: `display` プロパティが `block` に上書きされており、インラインの `style` 属性自体が要素から消えている事実を観測。
3. **原因特定**: ホスト環境のサニタイズ処理がインラインスタイルを削除していると特定。
4. **対策の実装**: `document.createElement` で要素を作り、`.style.cssText` でスタイルを直接指定する手法に切り替え。
5. **最終検証**: 再び `browser_subagent` で `display: flex` が保持されている「事実」を確認。
