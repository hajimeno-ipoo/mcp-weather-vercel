# 気温グラフの改良と降水表示の統一

## 概要
ウィジェットの気温グラフの範囲と意匠を改良し、降水確率アイコンの表記を統一します。また、降水データが正しく取得されているかを実データで検証します。

## 提案される変更点

### 気温グラフの改良
- `maxT` を `45`、`minT` を `-20` に変更。
- `.chart-area` の `height` を `320px` に拡大。
- `linearGradient` 内の 0度地点（`zeroOffset`）を計算し、「オレンジ」から「青」への急峻な切り替えを実装。

### 降水表示の統一
- 1時間ごとのカード内の降水確率アイコンを `💧` から `☔` に変更。
- 詳細パネル内の降水量ラベルを `☔` に変更。

## 検証計画
1. `browser_subagent` を使用し、極端な気温（-20°C, 45°C）を含むモックデータでグラフの描画を検証。
2. 雨が降っている地域の座標（シアトル等）を用いて API を直接呼び出し、`precipitation_sum` が 0 以外の値を返すことを証明。
