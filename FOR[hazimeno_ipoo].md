# MCP Weather Vercel プロジェクト概要

このプロジェクトは、**Open-Meteo API** を利用して、場所名や座標から最新の天気予報を取得し、リッチなUI（ウィジェット）で表示する **MCP (Model Context Protocol) サーバー** です。Vercel 上で動作するように設計されています。

## 主な機能
- **場所検索 (`geocode_place`)**: 日本語での場所名入力から、正確な緯度・経度とタイムゾーンを特定します。
- **天気予報 (`get_forecast`)**: 指定した場所の7日間の天気、気温推移グラフ、および詳細な1時間ごとの予報（気温・降水確率・アイコン）を表示します。
- **リッチ・ウィジェット**: OpenAI などのチャットインターフェース上で、視覚的に分かりやすいインタラクティブなウィジェットを提供します。

## 技術スタック
- **Vercel / Next.js**: サーバーレス環境での実行。
- **MCP SDK**: Model Context Protocol への準拠。
- **Vanilla JS + DOM API**: ウィジェットのレンダリング（ホスト環境の制限を回避するため、あえてDOM APIを直接操作しています）。

## 開発の心得 (エンジニアリング・ポリシー)
このプロジェクトでは、以下の原則を徹底しています：
1. **事実に基づいた調査**: 推測でコードを直さず、ブラウザのデバッグツールなどで「事実」を直接確認します。
2. **証拠による検証**: 実装後は必ず実際の表示を「目視」または「DOM検査」で確認し、証拠（スクリーンショット等）を残します。

## スキル
